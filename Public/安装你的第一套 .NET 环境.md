## 二、 设计理念（[design points](https://devblogs.microsoft.com/dotnet/why-dotnet/)）

> - Productivity is full\-stack with runtime, libraries, language, and tools all contributing to developer user experience\.
> - Safe code is the primary compute model, while unsafe code enables additional manual optimizations\.
> - Static and dynamic code are both supported, enabling a broad set of distinct scenarios\.
> - Native code interop and hardware intrinsics are low cost and high\-fidelity \(raw API and instruction access\)\.
> - Code is portable across platforms \(OS, chip architecture\), while platform targeting enables specialization and optimization\.
> - Adaptability across programming domains \(cloud, client, gaming\) is enabled with [specialized implementations](https://github.com/dotnet/designs/blob/main/accepted/2020/form-factors.md) of the general\-purpose programming model\.
> - Industry standards like [OpenTelemetry](https://opentelemetry.io/) and [gRPC](https://grpc.io/) are favored over bespoke solutions\.

## 三、 \.NET 组件（Components）

1. Runtime 也就是Run and Debug ，负责处理执行引擎、垃圾回收等一系列与运行时相关的内容

2. Libraries 负责提供编译时的核心代码（类似于 C 语言开头所用的头文件）

3. Compiler 即编译器（ [Roslyn](https://github.com/dotnet/roslyn) 是 C\# 和 VB 的专属编译器）

4. SDK and other tools 也就是辅助软件开发的一系列工具包（比如 [NuGet](https://learn.microsoft.com/en-us/nuget/what-is-nuget)）

5. App stacks 例如 [ASP\.NET](https://njupt-sast.feishu.cn/wiki/Z8kmwa4KkirKBckaV40cvZpxnGb)[ Core](https://njupt-sast.feishu.cn/wiki/Z8kmwa4KkirKBckaV40cvZpxnGb) 、[WinUI 3](https://njupt-sast.feishu.cn/wiki/Zy59wXUnjinmTWkTA49cIyTqnrb) 等

## 四、 \.NET 生态（Ecosystem）

> \.NET Core Its evolving and actively supported\.
> \.NET Framework Its actively supported, in maintenance\.  —— Microsoft

1. \.NET\(Core\) 也就是现代 \.NET。具有跨平台的特性以及[开源的仓库](https://github.com/dotnet)

2. \.NET Framework 原始 \.NET。仅仅服务于Windows

3. Mono 由Xamarin公司主持的社区，目的是打造一系列符合ECMA标准的\.NET工具

### 7、 Git 安装 （此方法将会安装整个 dotnet 仓库，适用于想要深入研究 dotnet 的玩家）

> 此处仅仅是搬运[dotnet仓库](https://github.com/dotnet/dotnet)内容
>
>

什么是 Git 请参考 [GitDocs](https://git-scm.com/docs)

* [ ] 克隆 dotnet 仓库` git clone ``https://github\.com/dotnet/dotnet`` path\-to\-your\-aim`或者对于SSH使用者`git clone ``git@github\.com``:dotnet/dotnet\.git path\-to\-your\-aim`

* [ ] 前往安装目录`cd path\-to\-your\-aim`

* [ ] 构建\.NET SDK

For Unix`\./build\.sh \-\-clean\-while\-building`

For Windows `\.\\build\.cmd \-cleanWhileBuilding`

* [ ] 最终会在 \./artifacts/assets/Release/ 目录下生成对应的压缩文件 dotnet\-sdk\-\*

* [ ] 拆包并安装 \.NET SDK

For Unix

`mkdir \-p $HOME/dotnet`

`tar zxf artifacts/assets/Release/dotnet\-sdk\-\* \-C $HOME/dotnet`

`ln \-s $HOME/dotnet/dotnet /usr/bin/dotnet`

For Windows

`mkdir %userprofile%\\dotnet`

`tar \-xf artifacts/assets/Release/dotnet\-sdk\-\* \-C %userprofile%\\dotnet`

`set \&\#34;PATH=%userprofile%\\dotnet;%PATH%\&\#34;`

## 1、 对软件安装

### （2）对 Visual Studio Code and Rider

参考对 CLI / 网页 安装

## 2、 对 CLI / 网页 安装

最安全的方法就是通过 winget 下载 Microsoft\.DotNet\.UninstallTool。

`winget install \-\-id Microsoft\.DotNet\.UninstallTool \-\-source winget`

运行如下命令完成卸载。

`dotnet\-core\-uninstall remove aim\-to\-uninstall`

最后顺手卸载卸载程序。

`winget uninstall \-\-id Microsoft\.DotNet\.UninstallTool`
